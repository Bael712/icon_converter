# SquareCropper Web 神版

**インストール不要・ブラウザだけで完璧な正方形切り取り＋ICOも1クリックで作れる神ツール**

今まで「512×512にしたいけどズレる…」「小さい画像がうまく拡大できない…」「ICO作るの面倒…」って泣いてた人、**もう泣かなくていい**。

これ一つで全部解決！！
## できること

- 256 / 512 / 1024 / 2048px 正方形切り取り（自由選択）
- 小さい画像でも**自動で枠ピッタリ拡大**
- 枠ドラッグで位置調整
- Shift＋ドラッグで画像移動
- ホイールでズーム
- PNG保存 → 即完成
- **ICO欲しい人 → 同梱の `make_ico.py` にPNGドロップで即512+256の最高品質ICO完成**

## 使い方（30秒）

1. ブラウザで開く → [https://あなたのユーザー名.github.io/square-cropper/](https://あなたのユーザー名.github.io/square-cropper/)
2. 「画像を選択」→好きな画像をドロップ
3. 枠をドラッグして位置調整
4. 「PNG保存」→ 完了！
5. ICO欲しい人は → 保存したPNGを `make_ico.py` にドラッグ＆ドロップ → 即ICO完成！
